# 手指神槍手 · Finger Gun

用手比成槍、對著鏡頭扣扳機射靶的瀏覽器小遊戲。純前端、零後端,基於 Google MediaPipe 手部追蹤。

## 玩法
- 👉 食指伸出來瞄準,準心跟著指尖
- 👍 大拇指立著 = 已上膛
- 👇 大拇指往下壓 = 扣扳機開火

定點靶,打掉一個換下一個;打中正中紅心得 2 分,邊緣 1 分。

## 技術
- 單一 `index.html`,無需建置步驟
- MediaPipe Hand Landmarker(在瀏覽器本機運算,鏡頭畫面不外傳)
- 需要 HTTPS 網址與鏡頭權限才能運作(本機直接開檔的 `file://` 不會給鏡頭權限,要部署或用 localhost)

## 部署
靜態網站,Vercel 直接 import 即可,無需任何環境變數或設定。
